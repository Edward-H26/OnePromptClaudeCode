# Deep Research

Use this skill for thorough research, comparative analysis, and source-backed reports.

## Repo Rules

- Use current web sources and cite them.
- Do not require `~/.claude.json`, global MCP setup, or global skill installs.
- Treat vendored deep-research content as background reference only.

## Workflow

1. Gather the minimum set of current, high-quality sources needed for the request.
2. Separate current external facts from repo-local observations.
3. Deliver findings with citations, assumptions, and remaining uncertainty.
