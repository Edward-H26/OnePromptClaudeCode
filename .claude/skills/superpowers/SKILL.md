# Superpowers

Use this wrapper when the user asks for Superpowers methods or when a task benefits from the locally enabled Superpowers plugin.

## Repo Rules

- Prefer Superpowers plugin skills when they are enabled locally.
- If the plugin is unavailable, use the repo-local equivalent workflow, such as `/orchestrate`, `/super-ralph`, `/review-staff`, or direct verification.
- Do not make plugin availability a hard blocker unless the user explicitly requires that plugin.

## Workflow

1. Match the task to the appropriate Superpowers workflow.
2. Use the plugin skill when available, otherwise use the closest repo-local workflow.
3. Verify outcomes with concrete evidence before reporting completion.
