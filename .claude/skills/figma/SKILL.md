# Figma

Use this wrapper when the user asks to inspect, create, or implement from Figma or FigJam.

## Repo Rules

- Prefer the Figma plugin when it is enabled locally.
- If the plugin is unavailable, ask for a concrete design artifact or continue from local assets when possible.
- Do not invent design details that should come from the Figma source.

## Workflow

1. Identify the Figma file, node, or design task.
2. Use the matching Figma plugin skill or MCP tool when available.
3. Verify generated code or diagrams against the retrieved design details.
